1. Description of Service

Claude Code Notifier is a notification system designed to work with Claude Code (Anthropic's command-line interface tool). The service consists of:

  • Claude Code Notifier (macOS): A menu bar application that monitors Claude Code sessions and delivers notifications when Claude requires user attention or permission approval.
  • Claude Code Notifier Companion (iOS/Android): Mobile companion applications that receive notifications from your Mac and enable remote responses to Claude Code prompts.
  • Cloud Relay Service: A server-side component that facilitates communication between your Mac and mobile devices.
